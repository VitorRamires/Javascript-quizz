<h2>JS Quizz </h2>
<h3>🎯 Objetivo</h3>
Este projeto foi criado com o objetivo de testar e consolidar conhecimentos em JavaScript através de questões cronometradas, com feedback visual imediato e registro detalhado de desempenho ao final.
<br />
<br />
<h3>🛠️ Construído com:</h3>
<ul>
<li>Javascript - linguagem base do projeto</li>
<li>React.js - Framework</li>
</ul>
<br />
<h3>✨ Funcionalidades:</h3>
<ul>
<li>⏱️ Timer de 15 segundos por questão</li>
<li>✅ Feedback visual da resposta correta e errada</li>
<li>📊 Tela de resultados com percentual de acertos, erros e questões puladas</li>
<li>🔁 Opção de reiniciar o quizz</li>
</ul>
<br />

<h3>🚀 Como rodar:</h3>

<pre><code>
# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
</code></pre>
